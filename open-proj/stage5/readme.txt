# How to Run  #
#			  #
###############


$ python main.py

~~ File containing elbow method ~~

		elbow.py

	- Note: The elbow chart is saved automatically
			as elbow.png in the current directory.

~~ File containing clustering ~~

		clusterer.py

~~ File containing regressor ~~

		regressor.py

~~ File containing data preprocessing ~~

		preprocessor.py

~~ How to change cluster value from elbow method ~~

	In order to do this you must manually change it in the code. In
	the clustering class there will be a variable called num_clusters 
	which you would change.
